# unixutils

Student implementation of zip(1), unzip(1), cat(1), and grep(1)

# Usage

## wcat

./wcat file1 file2 ...

## wgrep

./wgrep 'RegExp' file1 file2 ...

## wzip

./wzip output.zip file1 file2 ...

## wunzip

./wunzip input.zip

# Dependencies

* libpcre
* libzip
